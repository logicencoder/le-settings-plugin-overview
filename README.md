# LE Settings (public overview)

WordPress plugin that powers **site-wide configuration** on logicencoder.com: security, Telegram notifications, SEO-related controls, and operator diagnostics.

---

## Scope (this plugin only)

- Shared settings layer for the Logic Encoder theme ecosystem  
- Brute-force protection, monitoring hooks, maintenance/redirect helpers  
- Admin settings UI (not product catalogue, not crypto checkout)  

## Not the same as

| Plugin | Role |
|--------|------|
| [le-shop-plugin](https://github.com/logicencoder/le-shop-plugin) | App store catalogue + sync to Node backend |
| [le-crypto-app-store](https://github.com/logicencoder/le-crypto-app-store) | Telegram shop + payments on SOL |

Implementation (private): [le-settings-plugin](https://github.com/logicencoder/le-settings-plugin)
